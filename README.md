## Setup

Open your git bash and cd to the location you'd like to put your files the run the command below.

```console
git clone https://github.com/mohamednaser/TESTING-PRACTICE.git
```

## Testing

#### Get the dependencies needed for the game

`$ npm install`

#### Then run the test

`$ npm test`
